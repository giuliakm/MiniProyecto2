## Descripción del sistema

Este proyecto consiste en un sistema simple de consola desarrollado en Java que permite gestionar un pequeño refugio de animales.

El sistema permite registrar perros y gatos, mostrar los animales registrados, buscar un animal por su ID, cambiar el estado de un animal y ejecutar el sonido correspondiente a cada tipo de animal.

El objetivo del proyecto es aplicar los conceptos básicos de Programación Orientada a Objetos como clases, objetos, encapsulamiento, herencia, polimorfismo y manejo básico de excepciones.

---

## Funcionalidades del sistema

El sistema permite:

* Registrar un perro
* Registrar un gato
* Mostrar todos los animales
* Buscar un animal por ID
* Cambiar el estado de un animal
* Ejecutar el sonido de un animal
* Validar entradas incorrectas
* Manejar errores con try/catch

---

## Estructura del proyecto

El proyecto está organizado en las siguientes clases:

Animal.java
Clase base que contiene la información general de un animal.

Perro.java
Clase hija que hereda de Animal y define el comportamiento de un perro.

Gato.java
Clase hija que hereda de Animal y define el comportamiento de un gato.

Refugio.java
Clase que gestiona la lista de animales usando ArrayList y contiene la lógica del sistema.

Main.java
Clase principal que contiene el menú en consola y permite interactuar con el sistema.
